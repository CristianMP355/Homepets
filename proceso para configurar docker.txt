Haz esto (sin inventos, en este orden)

Reinicia Windows (importante).

Abre PowerShell (normal o admin) y corre:

wsl -l -v


Si te sale “no hay distribuciones instaladas” o algo así, instala una.

Instala Ubuntu (recomendado):

wsl --install -d Ubuntu


(o desde Microsoft Store: “Ubuntu”)

Asegura WSL2 como predeterminado:

wsl --set-default-version 2


Abre Docker Desktop y dale Restart si lo pide.

Qué deberías ver después

Docker ya no debería mostrar “WSL needs updating”.

El “Engine starting” debería pasar a “Running”.

Si quieres, te doy el diagnóstico exacto ya mismo

Mándame lo que te salga en:

wsl -l -v


Con eso te digo si te falta instalar Ubuntu o si Docker ya debería abrir normal.
